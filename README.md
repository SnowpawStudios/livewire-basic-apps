## Basic Livewire Apps

Seven simple apps that illustrate how to implement/solve common web application tasks. These include:

- A counter
- Simple calculator
- Todo list (basic crud functionality)
- Cascading dropdowns
- Basic search functionality
- Uploading images
- Real time form validation
